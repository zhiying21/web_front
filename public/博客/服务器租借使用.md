## <font style="color:rgb(15, 17, 21);">4.</font><font style="color:rgb(15, 17, 21);"> </font>**<font style="color:rgb(15, 17, 21);">部署你的图书馆系统</font>**
### <font style="color:rgb(15, 17, 21);">上传项目文件</font>
<font style="color:rgb(15, 17, 21);">bash</font>

```plain
# 方法一：使用 Git 克隆（推荐）
ssh root@服务器IP
cd /var/www/
git clone https://github.com/zhiying21/library.git

# 方法二：使用 SCP 上传
scp -r /本地项目路径/* root@服务器IP:/var/www/library/
```

### <font style="color:rgb(15, 17, 21);">安装环境依赖</font>
<font style="color:rgb(15, 17, 21);">bash</font>

```plain
# 更新系统
apt update && apt upgrade -y

# 安装 Python 和 pip
apt install python3 python3-pip -y

# 安装虚拟环境
pip3 install virtualenv

# 创建虚拟环境
cd /var/www/library
python3 -m virtualenv venv
source venv/bin/activate

# 安装项目依赖
pip install -r requirements.txt
```

## <font style="color:rgb(15, 17, 21);">5.</font><font style="color:rgb(15, 17, 21);"> </font>**<font style="color:rgb(15, 17, 21);">配置 Web 服务</font>**
### <font style="color:rgb(15, 17, 21);">安装和配置 Nginx</font>
<font style="color:rgb(15, 17, 21);">bash</font>

```plain
# 安装 Nginx
apt install nginx -y

# 创建 Nginx 配置文件
nano /etc/nginx/sites-available/library
```

### <font style="color:rgb(15, 17, 21);">Nginx 配置示例</font>
<font style="color:rgb(15, 17, 21);">nginx</font>

```plain
server {
    listen 80;
    server_name 你的服务器IP;
    
    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
    
    location /static/ {
        alias /var/www/library/static/;
    }
}
```

### <font style="color:rgb(15, 17, 21);">启用站点</font>
<font style="color:rgb(15, 17, 21);">bash</font>

```plain
ln -s /etc/nginx/sites-available/library /etc/nginx/sites-enabled/
nginx -t  # 测试配置
systemctl restart nginx
```

## <font style="color:rgb(15, 17, 21);">6.</font><font style="color:rgb(15, 17, 21);"> </font>**<font style="color:rgb(15, 17, 21);">启动应用</font>**
### <font style="color:rgb(15, 17, 21);">运行 Django 项目</font>
<font style="color:rgb(15, 17, 21);">bash</font>

```plain
cd /var/www/library
source venv/bin/activate

# 数据库迁移
python manage.py migrate

# 创建管理员账户
python manage.py createsuperuser

# 收集静态文件
python manage.py collectstatic

# 启动应用（生产环境建议用 Gunicorn）
python manage.py runserver 0.0.0.0:8000
```

### 1.3子查询

子查询是指嵌入在其他sql语句中的select语句，也叫嵌套查询。

#### 1.3.1单行子查询

```mysql
SELECT name from result where math>(SELECT AVG(math) FROM result);
```

#### 1.3.2多行子查询

- **返回结果**：子查询返回多行（一行或多行）。

- 常用运算符：

  - `IN/NOT IN`：检查值是否在子查询结果集中。

    ```mysql
    CREATE TABLE company(
    id int UNSIGNED PRIMARY KEY,
    department SET('技术部','市场部','销售部'),
    employees VARCHAR(20)
    );
    INSERT INTO company VALUES(1,'技术部','张三')
    INSERT INTO company VALUES(2,'市场部','李四')
    
    select * FROM company WHERE id IN 
    (SELECT department FROM company WHERE department in ('技术部','销售部'));
    ```

  - `ANY/SOME`：与子查询结果中的**任意一个值**比较。

    ```mysql
    select * FROM result WHERE math > any
    ```

  - `ALL`：与子查询结果中的**所有值**比较。

#### 1.3.3多列子查询

单行子查询是指子查询只返回单例；多行子查询是指返回单列多行数据，都是针对单列而言的，而多列子查询则是指查询返回多个列数据的子查询语句。

例子：

```mysql
SELECT name FROM result WHERE (math,english)=(SELECT math,english FROM result where id = 1) AND id <> 1;
```

#### 1.3.4在from 子句中使用子查询

子查询语句出现在from子句中，把子查询当作一个临时表看。

例子：显示出数学分高于数学平均分的人名

```mysql
SELECT name FROM result,(SELECT AVG(math) Math FROM result ) relt WHERE result.math>relt.Math;
```

1.3.5合并查询

在实际应用中，为了合并多个select的执行结果，可以使用集合操作符union，union all

1.3.5.1union

用于取得两个结果集的并集。当使用该操作符时，会自动去掉结果集中的重复行。

例子：找出数学成绩大于70或语文大于30

```mysql
SELECT name,math,chinese FROM result WHERE math>70 UNION 
SELECT name,math,chinese FROM result WHERE chinese>30;
```

1.3.5.2union all

用于取得两个结果集的并集。当使用该操作符时，不会自动去掉结果集中的重复行。

```mysql
SELECT name,math,chinese FROM result WHERE math>70 UNION ALL
SELECT name,math,chinese FROM result WHERE chinese>30;
```

## 2内外连接

### 2.1内连接

内连接实际上是利用where语句对两种表的形式进行筛选，我们前面学习的查询都是内连接，也是在开发中使用的最多的连接查询。

语法：

```mysql
SELECT 字段 FROM 表1 INNER JOIN 表2 ON 连接条件 AND 其他条件;
```

例子：

```mysql
SELECT t1.sex,t1.name, result.math FROM t1 JOIN result ON t1.name = result.name AND t1.name='张三';
```

### 2.2外连接

外连接分为左外连接和右外连接。

#### 2.2.1左外连接

联合查询，左侧的表完全显示出来。

语法：

```mysql
SELECT 字段名 FROM 表名1 LEFT JOIN 表2 ON 连接条件;
```

例子：

```mysql
SELECT t1.sex,t1.name, result.math FROM t1 LEFT JOIN result ON t1.name = result.name AND t1.name='张三';
```

联合查询，右侧的表完全显示出来。


例子：
